1. Установить Python 3.9.13
2. Установить VSCode
3. Открыть проект в VSCode и установить все необходимые библиотеки. Это делается при помощи команды: pip install -r requirements.txt
4. Создать файл .env на основке .env_example с переменными окружения и записать туда Ваш токен chatgpt.
5. Запустить веб-приложение при помощи команды: uvicorn main:app --reload
